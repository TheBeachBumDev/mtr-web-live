Routers

Register edge routers with SSH access and tie each one to an IPAM location (e.g. Oukaapse HS). Passwords are stored encrypted in the database and are never shown again in the UI; use them later from server-side automation.

Import routers from a text file

Put the IPAM location first on each line (exact name or numeric id) when each router has its own site, or pick a default location in the form for every line. SSH user, password, and port in the form apply to any column you leave out; you can also set them per line.

Optional first column(s): location name (multi-word names work when separated by tabs or commas) or numeric location id. Then: router_host · host user · host user password · host user password port — tabs, commas, pipes, or spaces (no spaces inside passwords). If you do not use a location per line, select a default Location (IPAM) in the form. Blank lines and # lines are skipped.
