Location Sync

Syncs active Splynx customers (same customer_id as Field Tech / IPAM). Multiple active internet services become multiple rows (keyed by service login / id from Splynx bulk internet-services API). Pagination uses 100-record API pages by default (override with SPLYNX_CUSTOMER_PAGE_SIZE) with a short pause between requests. MAC is taken from online sessions when present, otherwise last session / list / optional deep lookups (see env). Use Cross-reference to read active PPP sessions on a chosen edge router, match MACs to this cache, and set Location to that router’s site (from Routers / IPAM). Antenna IP in IPAM is left to field techs; it is not changed here.
